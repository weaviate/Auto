NearVector

You can use the `nearVector` operator to find objects most similar to an input vector (e.g. `[-0.368, 0.1397, ... , 0.0971]`).

```graphql
{
  Get {
    JeopardyQuestion (
      limit: 2,
      nearVector: {
        vector: [-0.0125526935, -0.021168863, -0.01076519, ...]
      }
    ) {
      question
      answer
      _additional {
        distance
      }
    }
  }
}
```
